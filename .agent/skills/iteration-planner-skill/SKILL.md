# Iteration Planner Skill

## Goal
WBS とガントをもとに、今回の iteration の対象範囲を明確にする。

## Instructions
1. `03_planning/wbs.md` と `03_planning/gantt.md` を読み、次に着手すべきタスク群を選ぶ。
2. iteration goal、対象機能、予定工数、AI協働予定工数を整理する。
3. acceptance criteria と紐づける。
4. `08_reports/master-report.md` に iteration 計画を追記する。

## Output Targets
- `08_reports/master-report.md`

## Constraints
- iteration 範囲を広げすぎない。
- acceptance criteria に紐づかない作業を優先しない。
