# Builder Skill

## Goal
iteration で選ばれた機能を実装し、成果を記録する。

## Instructions
1. 対象要件と implementation scope を確認する。
2. 実装する。
3. 変更ファイル、変更概要、未解決事項を整理する。
4. `04_implementation/phase-XX/implementation-report.md` を更新する。

## Output Targets
- `04_implementation/phase-XX/implementation-report.md`

## Constraints
- スコープ外の変更は明記する。
- 実装内容を記録せずに完了扱いにしない。
