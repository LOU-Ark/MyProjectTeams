# 🔱 Aurora Project Rules: Master Index

## 1. 運用原則
本プロジェクトの全エージェントは、以下の優先順位に従って行動せよ。
1. **Governance (GVN)**: プロジェクトの存立、納品、構造、および意思決定プロセス。
2. **Quality (QLT)**: 信頼性とエビデンスの担保、インシデント管理。
3. **Engineering (ENG)**: 効率的な実装、資産管理、ドキュメントの標準化。
4. **Visual (VIS)**: ブランドの一貫性、Mermaid および UI の美学。
