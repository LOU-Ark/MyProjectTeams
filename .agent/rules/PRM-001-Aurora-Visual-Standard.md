# Team Protocol: PRM-001 - Premium Visual Standard

**カテゴリ**: UI/UX 品質基準
**適用範囲**: 全プロジェクトのドキュメント、ダッシュボード、成果物ビューアー

## 1. 空間の定義
「情報の価値は、その周囲의余白に依存する」という原則に基づき、以下のマージンを適用する。
- **Outer Padding**: 40px (Viewport 端とメインコンテンツの最小距離)
- **Safe Zone**: ツールチップ表示時は境界から 20px 以上の距離を保つ。

## 2. 形状の定義
親しみやすさと洗練の融合を図る。
- **Border Radius**: 30px (主要コンテナ)
- **Bar Radius**: Capsule (100% Round) - タスクバー等の要素。

## 3. ハイブリッド描画基準 (0-test14 Charter Mode)
情報の性質に応じ、表示モードを自動選択すること。
- **Portal Mode (Slide)**: 密度と可読性を優先。1日の幅を固定(85px+)し、横スクロールによる詳細閲覧を許容する。
- **Presentation Mode (Full Fit)**: 俯瞰と要約を優先。別タブ等での表示時は、ビューポートの縦横解像度に合わせ、全情報をスクロールなしでフィットさせる。

## 4. インタラクションの定義
「知りたいときに、そこにある」情報を実現する。
- **Contextual Hover**: タスク等の重要なグラフィック要素は、ホバー時に詳細情報を引き出し、画面端での「はみ出し防止ロジック」を用いて表示すること。
- **Visual Feedback**: 全画面アイコン (⛶) を備え、プレゼンテーションへの即応性を確保する。

---
**遵守義務**: 本プロトコルに違反する表示は Aurora Brand において「未完成」と見なされる。
