# My Project Teams (.agent)

このリポジトリは、自律型AIエージェントによるプロジェクト実行・管理のための環境構成ファイル（`.agent` ディレクトリ）を管理しています。

## 概要 (Overview)
Aurora 基準 (VIS-004) に則り、要件定義から計画、実装、納品までの各フェーズを自動で遂行・検証するためのワークフロー、プロンプト、アセットが格納されています。

## ディレクトリ構成
- `workflows/`: 各フェーズの実行定義ファイル
- `templates/`: マークダウンやレポートなどの出力用ひな形
- `assets/`: ポータル等で使用されるCSSやスクリプトなどの共通アセット群

## 役割 (Roles)
この設定は、ユーザーとエージェントが協調して「仕組み（システム）」で課題を解決し、常に再現性のある最高品質の成果物を生み出すためのガバナンス基盤として機能します。
